display y visibility

display es es una propiedad importante para controlar estructuras. Cada elemento
tiene un valor de display por defecto dependiendo de que elemento sea. El valor por
defecto para la mayoria de los elementos es block (bloque) o inline. Un elemento que es
block se le dice block-level.

Block: el div es elemento block estandar. Un elemento block empieza en una nueva linea y se estira
hasta la derecha e izquierda tan lejos como pueda ( a menos que se especifique lo contrario)
Otros elementos comunes son p, form, header, footer, etc

inline: el span es ele elemento estandar. Un elemento inline puede contener
algo de texto sin interrumpir el flujo de parrafo. El elemento a es el inline mas comun


none: otro valor comun de display es none. 
Visibility: hidden

cambiar div a display:inline para ver el efecto
